# Let Em Speak!
Audio Isolation of speakers from a debate of speakers.

## Getting started

### Prerequisites
- Tensorflow >=2.0
- OpenCV >=
- Matplotlib >=
- MoviePy >=

### Installation
```
git clone https://github.com/kv3n/letemspeak.git
```

### Usage
```
py letmespeak.py
```
